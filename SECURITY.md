# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in this project, please report it to us privately via the private vulnerability reporting feature on GitHub. 

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.
